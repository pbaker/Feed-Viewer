About

This application is under development. When finished, it will provide an administrative dashboard to the Boxee video channel repository hosted by channelvision.tv

This "page" is built using the EXT javascript framework. The application is based on code from the EXT sample 'feed-viewer' -- a RSS feed reader.

It has been modified as follows:

Added support for XML namespaces, i.e., xmlns:somespace='www.somespace.com/rss-specifications
Auto-detects 'boxee' namespace tags (boxee:sometag)and displays the Boxee logo for detected feeds
Reads Boxee rss/xml tags to extract metadata, including the URLs of video feeds.
Fully-integrated support for the FloPlayer open-source video player.
Plays .mp4 and .flv video feeds using the built-in video player.
XML source viewer (text output)
XML source viewer (prettified)
Added support for external Brightcove player
